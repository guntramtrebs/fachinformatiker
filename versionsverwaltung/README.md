# Versionsverwaltung

## Ziele
- Protokollierung (wer, wann, was, warum)
- Wiederherstellung bestimmter Projektstände
- Versionierung / Archivierung
- Koordinierung von Zugriff / Berechtigung
- Koordinierung paralleler Entwicklung
